# Telegram MTProto Proxy

This project provides a simple way to run a **Telegram MTProto Proxy server** using **Docker on a Linux server**.

The proxy allows Telegram users to connect when direct access to Telegram is restricted.

It works on most environments including:

- VPS servers
- Cloud providers (Google Cloud, AWS, Azure, etc.)
- Dedicated Linux servers

⚠️ This proxy is designed **only for Telegram** and does not provide full internet access.

The setup is lightweight and can run on small servers with minimal resources.
